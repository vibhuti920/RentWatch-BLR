# RentWatch-BLR
A community analytics platform for Bengaluru tenants to report and resolve rental disputes.

## What it does
A full-stack discussion and complaint forum covering 4 complaint categories — deposits, maintenance, agreements, and disputes. Supports upvotes and threaded responses for community-driven resolution.

## Tech Stack
Python, Flask, SQLite, HTML

## Features
- Relational database schema for structured complaint storage
- Trend surfacing for rental disputes across 100+ users
- Community upvoting and threaded discussion system
